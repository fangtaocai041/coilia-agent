## 🧩 这个项目是什么
```
刀鲚（*Coilia nasus*），"长江三鲜"之首。每年春夏之交，它们从海里游回长江产卵。但过度捕捞和水工建筑让它们的洄游路线越来越艰难。
```
这个项目通过耳石微化学（Sr/Ca 比值）重建每一条鱼的洄游历史——它在哪里出生、在哪里长大、在哪里产卵。每一片耳石，都是一本打开的生命日记。
```
> 赫拉克利特说：人不能两次踏进同一条河流。
>
> 我们说：但一条刀鲚可以——只要河流还在。
```
